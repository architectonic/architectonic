# Changelog

## 0.3.0

- Add deterministic `recommend` guidance that can select no install, one standalone layer, a named profile, or an exact custom composition.
- Make constitution, identity, project, knowledge, and other useful concerns directly installable without the full organization.
- Separate ordinary knowledge from living-knowledge maintenance and add explicit selection criteria.
- Add `llm-wiki`, `second-brain`, `agent-team`, `loop-system`, and `graph-system` profiles.
- Add feature-specific local scaffolds for source-first wikis, second brains, loops, and graph policy.
- Add rebuildable JSON and DOT graph projections from explicit Markdown links and wikilinks.
- Add graph-quality findings for broken links, ambiguous links, orphans, and gravity wells.
- Encode lessons from OKF, LLM Wiki, GBrain, second-brain methods, loop engineering, graph engineering, and public Agent Skills without making any external runtime mandatory.
- Clarify that Obsidian and Graphify are optional visualization or analysis surfaces and that Workframe is an optional execution environment.
