# Interoperability

Architectonic is file-native and runtime-neutral. It defines durable boundaries and leaves execution, retrieval, visualization, and storage replaceable.

## OKF-compatible practice

Architectonic uses ordinary Markdown and structured metadata so documents can participate in OKF-style progressive disclosure:

```text
index.md        compact entry and routing surface
frontmatter     type, title, status, provenance, risk, or local extensions
log.md          accepted durable changes
linked files    deeper concepts and procedures
```

The system does not depend on one parser or hosted catalog.

## Obsidian

Obsidian may visualize local Markdown, backlinks, tags, and wikilinks. It is a view and authoring environment, not the authority model or knowledge-maintenance protocol.

## Graphify and graph infrastructure

Graphify or another graph pipeline may extract entities, infer relations, identify communities, or generate graph indexes. Generated edges must remain attributable and reviewable. Canonical source files remain above the projection.

## Retrieval and databases

A Git/Markdown system of record may be accompanied by:

- lexical search;
- vector retrieval;
- graph traversal;
- structured databases;
- external dataset warehouses;
- hybrid reranking and query planning.

These are replaceable access paths. Scores and generated summaries are not evidence by themselves.

## Public skills

External Agent Skills can be copied, vendored, or referenced with provenance. Architectonic’s skills layer adds local trust review, risk classification, verification expectations, and authority constraints. Compatibility does not imply approval.

## Workframe

Workframe may execute governed work as runs, leases, approvals, artifacts, and ledger entries.

```text
Architectonic  durable purpose, authority, projects, skills, knowledge, agents, and upkeep
Workframe      optional execution, scheduling, claims, approvals, and runtime evidence
```

Either system can exist without the other. Together, Architectonic supplies the durable organizational contract and Workframe supplies an execution environment.
